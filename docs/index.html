🚀 快速开始

1 克隆项目

git clone https://github.com/henryhuoo/openclawAPP.git

2 安装依赖

npm run install:all

3 运行本地插件

cd openclaw-plugin && npm run dev

4 访问 Web 客户端

打开浏览器访问部署好的客户端地址，开始对话！
